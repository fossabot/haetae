# Getting Started

You can incrementally adopt Haetae for new or existing projects.
Haetae is simple and flexible.
It does not matter if your project is monorepo or not.

## Installation

```sh
npm install haetae
```

<Callout>
  **Are you developing a library(e.g. plugin) for Haetae?** <br />
  You can install `@haetae/core`, `@haetae/utils`, `@haetae/git`, `@haetae/javascript`,
  `@haetae/cli` independently. Note that the package `haetae` include all of them.
</Callout>
