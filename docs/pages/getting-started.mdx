# Getting Started

Haetae is simple and flexible.<br/>
You can incrementally adopt it for existing projects.<br/>
It also works well with monorepos.

## Installation

```sh
npm install haetae
```

<Callout>
  **Are you developing a library(e.g. plugin) for Haetae?** <br />
  You can install [`@haetae/core`](apis/haetae-core), [`@haetae/utils`](apis/haetae-utils),
  [`@haetae/git`](apis/haetae-git), [`@haetae/javascript`](apis/haetae-javascript),
  [`@haetae/cli`](apis/haetae-cli) independently. Note that the package [`haetae`](apis/haetae)
  include all of them.
</Callout>
