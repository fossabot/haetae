# `@haetae/javascript`

`@haetae/javascript` provides features for javascript ecosystem.<br/>
If your project do not use javascript-related languages, you don't need this package.

<Callout>
  CJS (CommonJS), ESM (ES6 module), AMD, Typescript, JSX, TSX, Sass, Stylus,
  Less, CSS (PostCSS), nodejs-only modules, browser-only modules are **ALL
  SUPPORTED.**
</Callout>
