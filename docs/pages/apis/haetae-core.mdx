# `@haetae/core`

`@haetae/core` contains every core feature for Haetae.<br/>
It handles config file and store file.<br/>
It also evaluates `env` and executes `run` of your command.<br/>
It provides every core feature Haetae needs.<br/>
If you want to use Haetae by programatic api, you probably need this package.
