# Haetae

**Haetae** is a framework or runner for incremental tasks.

No matter whether it's lint, test, build, it can help you.

It's NOT an alternative to eslint, jest, tsc, rome, turborepo, nx, etc.

Haetae is for

Here's what you will get in 1 minute:

```sh
# this runs test for only affected files
$ npx haetae myTest
```
